optim_DSSA - Direct Search Simulated Annealing

Calling Sequence

[xmin,x_history] = optim_DSSA(f,U,L,ItMX)

Parameters

  • f : the objective function
  • U : a vector of upper bounds for x
  • L : a vector of lower bounds for x
  • ItMX : maximum number of iterations
  • xmin : the best solution found so far
  • x_history : a list of all the points generated by the optimization method

Description

This optimization method is an hybrid between simulated annealing and nelder-mead local search.
