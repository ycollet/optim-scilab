optim_pt - Parallel tempering

Calling Sequence

[x_best,x_history] = optim_pt(x0,pt_f,ItMax,T,pt_proba,Log,neigh_func,param_neigh_func)

Parameters

  • x0 : a starting point for the parallel tempering method
  • pt_f : the objective function
  • ItMax : maximum number of iterations
  • T : a vector of temperatures
  • pt_proba : the probability of increasing or decreasing the temperatures (default 0.05)
  • Log : a boolean to allows to display some informations during the execution of the optimization method
  • neigh_func : a neighborhood function
  • param_neigh_func : a list of parameters to be sent to neigh_func
  • x_best : the best solution found so far
  • x_history : a list of all the points computed by the optimization method

Description

This method implements the parallel tempering which correspond to several simulated annealing in parallel with a random reinitialization of some of the simulated annealing.
