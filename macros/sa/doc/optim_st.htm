optim_st - Simulated tempering

Calling Sequence

[x_best,temp_list,x_history] = optim_st(x0,st_f,ItMax,T,st_proba,Log,neigh_func,param_neigh_func)

Parameters

  • x0 : a starting point for the simulated tempering method
  • st_f : the objective function
  • ItMax : maximum number of iterations
  • T : a vector of temperatures
  • st_proba : the probability of increasing or decreasing the temperatures (default 0.05)
  • Log : a boolean to allows to display some informations during the execution of the optimization method
  • neigh_func : a neighborhood function
  • param_neigh_func : a list of parameters to be sent to neigh_func
  • x_best : the best solution found so far
  • temp_list : a vector of the temperatures selected by the simulated tempering
  • x_history : a list of all the points computed by the optimization method

Description

This method implements the simulated tempering which is a simulated annealing with an "automatic" tuning of the temperature.
