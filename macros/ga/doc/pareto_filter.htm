pareto_filter - A function which extracts a Pareto frontier from a given data set

Calling Sequence

[F_out,X_out,Ind_out] = pareto_filter(F_in,X_in)

Parameters

  • F_in : a vector of objective functions values (dimension: nbpoints x nbobjfunc)
  • X_in : a list of points coordinate which corresponds to F_in (optional parameter)
  • F_out : a vector of objective functions values which belong to the Pareto frontier
  • X_out : a list of points coordinate which corresponds to F_out (optional parameter)
  • Ind_out : a vector of indexes of the points in F_in (and X_in) which belongs to the Pareto frontier (F_out = F_in(Ind_out,:))

Description

A function which extracts points which belong to the Pareto frontier.
