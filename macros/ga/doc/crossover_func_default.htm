crossover_func_default - A GA function which performs a crossover between 2 selected individuals

Calling Sequence

[Crossed_Indiv1,Crossed_Indiv2] = crossover_func_default(Indiv1,Indiv2)

Parameters

  • Indiv1 : The first individual on which we will perform the crossover
  • Indiv2 : The second individual on which we will perform the crossover
  • Crossed_Indiv1 : The first child (a result of the crossing over of Indiv1 and Indiv2)
  • Crossed_Indiv2 : The second child (a result of the crossing over of Indiv1 and Indiv2)

Description

A GA function which performs a crossover between 2 selected individuals

Examples

    // A blending crossover for continuous optimization
    function [Crossed_Indiv1, Crossed_Indiv2] = crossover_func_default(Indiv1, Indiv2)
      mix = rand(1,1);
      Crossed_Indiv1 =     mix*Indiv1 + (1-mix)*Indiv2;
      Crossed_Indiv2 = (1-mix)*Indiv1 +     mix*Indiv2;
    endfunction
