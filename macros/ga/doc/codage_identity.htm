codage_identity - A GA function which encodes an individual into an intermediate representation

Calling Sequence

pop_out = codage_identity(pop_in,direction)

Parameters

  • pop_in : the population of individuals to transform. Be careful: pop_in is a list
  • direction : a string which indicates the type of process we ant to perform: 'code' or 'decode'
  • pop_out : the transformed population of individuals. Be careful, pop_out must be a list

Description

A GA function which encodes an individual into an intermediate representation. This function implement an identity transformation which allows the GA to behave like an evolution strategy.

Examples

  function [pop_out] = codage_identity(pop_in, direction)
  if (direction=='code') then
    pop_out = pop_in;
  elseif (direction=='decode') then
    pop_out = pop_in;
  else
    error('codage_identity: wrong direction');
  end
  endfunction
