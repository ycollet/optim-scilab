init_func_default - A GA function which produces an initial population of individuals

Calling Sequence

Pop_init = init_func_default(popsize)

Parameters

  • popsize : The size of the population we want to produce
  • Pop_init : The initialized population. Be careful: Pop_init must be a list (Pop_init = list())

Description

A GA function which produces an initial population of individuals

Examples

    // We produce a population of vectors of size 2 between -2 and +2
    function Pop_init = init_func_default(popsize)
      Dim = 2;
      Max = 2*ones(1,Dim);
      Min = -2*ones(1,Dim);

        Pop_init(i) = (Max - Min).*rand(size(Max,1),size(Max,2)) + Min;
      end
    endfunction
